> python3 -m venv .venv
> source .venv/bin/activate
> python -m pip install django~=4.0.0

> django-admin startproject mysite .
> python3 manage.py runserver

> python manage.py startapp polls
# need to create urls.py in the polls directory
# need to add "polls.apps.PollsConfig", in INSTALLED_APPS in mysite/settings.py

# do this when you add/cahnge models
> python3 manage.py makemigrations polls
> python manage.py sqlmigrate polls 0001 (will display sql that will be executed)
> python manage.py migrate

> python mange.py shell (to manually add data using python, can not use simple python)

> python manage.py createsuperuser


In order to create django app in a subfolder:

> mkdir apps
> cd apps
> touch __init__.py
> python ../manage.py createapp rentals
In RentalsConfig change name from rentals to apps.rentals
Add this line in main urls:
> path("", include("apps.rentals.urls")),
